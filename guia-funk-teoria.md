## 4. COMO CRIAR MELODIA EM D#M — PASSO A PASSO NO PIANO ROLL

### Estrutura da melodia funk:

**1. ESQUELETO MELÓDICO (usando a escala de D#m)**

Melodia base (simples, repetitiva):
```
Notas: D#4(63) — F4(65) — F#4(66) — G#4(68) — F#4(66) — F4(65) — D#4(63)
```
MIDI no piano roll: `63 - 65 - 66 - 68 - 66 - 65 - 63`
Duração: semínimas ou colcheias

**2. PADRÃO DE REPETIÇÃO (típico do funk)**

A melodia funk repete o mesmo motivo com pequenas variações:
```
Motivo A: 63 - 66 - 68 - 66     (D# - F# - G# - F#)
Motivo A': 63 - 66 - 70 - 66    (D# - F# - A# - F#) — variação
Motivo B: 68 - 71 - 75 - 71     (G# - B - D# - B) — resposta
```

**3. CALL AND RESPONSE (pergunta e resposta)**

```
Call:  D#4(63) — F#4(66) — G#4(68) — G#4(68)   (2 compassos)
Resp:  F#4(66) — F4(65) — D#4(63) — D#4(63)     (2 compassos)
```

**4. RIFF DE 3 NOTAS (muito comum no funk)**

Pegue 3 notas da escala e crie um loop:
```
Ex1: 63 - 66 - 68  (D# - F# - G#) — ascendente
Ex2: 70 - 66 - 63  (A# - F# - D#) — descendente
Ex3: 66 - 68 - 71  (F# - G# - B)  — saltando
```

## 8. COMO DESCOBRIR O TOM DE UMA MÚSICA (APLICADO)

**Método prático para funk:**
1. Toque a nota D# (MIDI 63) junto com a música — se "casar", pode ser D#m
2. Toque o acorde D#m (63-66-70) — se soar como "casa", a música está em D#m
3. Verifique: a última nota do baixo/808 geralmente é a tônica (D#=51)
4. As notas pretas no piano que mais aparecem = escala de D#m (D#, F, F#, G#, A#, B, C#)

**Atalho: conte os sustenidos**
- D#m tem 6 sustenidos (F#, C#, G#, D#, A#, E#)
- Relativo maior: F#M (mesmo campo harmônico)

## 10. RESUMO — ATALHOS MENTAIS

1. **Tônica da melodia funk em D#m:** use D#4 (63) como nota de repouso
2. **Notas "seguras" para melodia:** D#4(63), F#4(66), G#4(68) — as 3 notas do acorde D#m
3. **Tensão melódica:** A#4(70) e B4(71) — ótimas para criar "choro" ou "puxada"
4. **808 sempre na raiz:** D#(51/39) para D#m, A#(58/46) para A#m, G#(56/44) para G#m
5. **Melodia funk = repetição com micro-variações:** não mude muita coisa, repita o motivo principal
6. **Contraste:** notas graves (51-58) para base, notas médias (63-71) para melodia, notas agudas (75-85) para acentos
7. **Toque "fora da grade" levemente:** funk brasileiro tem swing, não seja robótico
